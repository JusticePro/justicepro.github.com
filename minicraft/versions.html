Versions

Minicraft+

Minicraft+ was created based on the original Minicraft made by Marcus Persson. Minicraft+ was created by David.b and Dillyg10. Later picked up by Chris J and is now being developed by JusticePro. For more information or download go here.

MysticCraft 

MysticCraft was created by GameCreator2004. It contains many features. It is a mod of the original Minicraft and is not a base of Minicraft+. For more information or download go here.

Minicraft Delux 

Minicraft Delux is a class mod from 2012 and has not been updated in a while but still makes it here. For download go here.

Minicraft Original 

The original Minicraft game that was created by Marcus Persson in 48 hours for ludum dare 22. For download go here.

Miniventure

This is game that takes many features from Minicraft and Minicraft Mods. It is inspired by the game. It was created by Chris J after he abandoned Minicraft+. For more information or download go here .
